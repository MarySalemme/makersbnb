# Makersbnb
A web application that allows users to list spaces they have available, and to hire spaces for the night.


**User stories**
```
As a user,
So that I can list a new space,
I want a sign-up feature.

As a user,
So that I can access my account,
I want to login.

As a user,
So that I can leave the site securely,
I want to logout.

As a user,
So that I can list many spaces,
I want an index for spaces.

As a user,
So that I can promote my space,
I want to be able to name and describe the space.

As user,
So that I can make buck,
I want to be able to price my space.

As a user,
So that my guests know when they are entitled to stay,
I want to display a range of dates when the space is free.

As a signed-up user,
So that I can use a space,
I want to be able to hire a space for a night.

As a space-hosting user,
So that I know who want to book a space,
I want to be able to approve bookings.

As a space-hosting user,
So that my property doesnâ€™t get overbooked,
I want to be able to display unavailable dates.

As a space-hosting,
So that the space can still be booked while any request are unconfirmed,
I want the space to appear as free between request and confirmation.

```
